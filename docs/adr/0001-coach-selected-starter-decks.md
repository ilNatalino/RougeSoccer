# Coach-selected starter decks

The player chooses an Allenatore before the first Partita to select one of three Mazzi di Partenza, each built around a football Stile di Gioco and exclusive Carte Identitarie. The Allenatore does not add passive powers, alter shared cards, or constrain later rewards; variety should come from the initial deck composition so the game remains coherent as a football deckbuilder rather than becoming a class-based RPG.
